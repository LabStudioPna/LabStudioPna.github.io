LABSTUDIO — KIT DE MARCA
=========================

EL CONCEPTO
-----------
El isotipo son 4 arcos en degradé (coral → violeta) girando alrededor de un
punto central: representan los 4 "canales" del estudio (web, dron, edición,
IA) emitiendo desde una misma señal. Es el mismo lenguaje visual ya usado
en la landing page (CH·01–CH·04, "señal activa", el puntito que pulsa
junto al logo en el header).

QUÉ ARCHIVO USAR EN CADA CASO
------------------------------

FAVICON DEL SITIO WEB
  Subí estos 5 archivos a la misma carpeta donde está tu index.html:
    favicon.ico
    favicon-16x16.png
    favicon-32x32.png
    apple-touch-icon.png
    site.webmanifest
    android-chrome-192x192.png
    android-chrome-512x512.png
  El index.html ya tiene los <link> necesarios en el <head>, así que
  con subir los archivos alcanza — no hay que tocar más código.

FOTO DE PERFIL (Instagram, WhatsApp Business, Google Business)
  social-avatar-1080.png
  Ya tiene el margen calculado para que el recorte circular de cada
  red no le corte el diseño.

LOGO PARA EL HEADER/FOOTER DEL SITIO O PARA DOCUMENTOS
  logo-horizontal-dark-bg.svg   → para fondos oscuros (como tu web actual)
  logo-horizontal-light-bg.svg  → para fondos blancos (PDFs, presupuestos,
                                   presentaciones, papelería)
  Son vectoriales (SVG): se pueden agrandar a cualquier tamaño —cartel,
  remera, vinilo del auto/drone— sin perder calidad. El texto ya está
  convertido a trazos, así que se va a ver igual en cualquier compu,
  tenga o no instalada la tipografía Space Grotesk.

SOLO EL ISOTIPO (sin texto)
  icon-master.svg / icon-512.png   → versión con fondo, para usar como
                                      ícono de apps, stickers, avatares
  icon-transparent.svg             → el anillo solo, para poner sobre
                                      cualquier color de fondo
  icon-mono-light.svg              → versión blanca lisa (para fondos
                                      oscuros, grabado, una sola tinta)
  icon-mono-dark.svg               → versión negra lisa (para fondos
                                      claros, sellos, una sola tinta)
